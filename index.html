TEAM FIRE BOY WATER GIRL

Our project is focused on developing reinforcement learning based agent with an Advantage Actor-Critic Model to play a single-player, 2D platformer, pre-existing game named ‘Zed’.
‘Zed’ is a goal-based video game, where the agent has to start at one point, and to complete the level, must reach the end while collecting coins and avoiding obstacles.
